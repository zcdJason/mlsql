接口访问Token设置

MLSQL Engine 对外提供HTTP服务默认是没有权限验证的，原因是我们假设他是在一个内网安全环境，只有特定应用才能访问该服务。MLSQL Engine从2.0.0版本开始对接口访问提供了Token验证。

设置启动Token

在spark-submit 命令行里，添加

--conf spark.mlsql.auth.access_token=your-token-string
