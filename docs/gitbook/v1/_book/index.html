MLSQL Engine产品简介

简介

MLSQL Engine 是MLSQL语言的官方实现，它同时也是一个分布式执行引擎。 搭配MLSQL Console 使用，可以获得良好的交互体验。

MLSQL语言是专门面向大数据和AI而设计的一门语言。在MLSQL中，SQL语言是第一公民，可内嵌Python支持,具体可参考MLSQL语法指南。

MLSQL Engine具有的特点：

  1. 支持单机或者分布式运行
  2. 可运行于K8s, Yarn, Mesos等通用资源管理环境
  3. 支持 Rest/JDBC 等交互接口
  4. 支持大部分大数据存储类应用，如HDFS,Hive,DeltaLake,MySQL等等
